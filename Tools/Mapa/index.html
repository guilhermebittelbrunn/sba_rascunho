CineRec

Trabalho acadêmico com tema de streaming de filmes

Instalação e inicialização

Use seu package manager para instalar as dependências

Front-end (client-side)

cd front
yarn
yarn dev

Back-end (server-side)

cd back
yarn
yarn server

Lembre-se de criar uma conta no The Movie Database para definir uma key e acessar as recomendações
